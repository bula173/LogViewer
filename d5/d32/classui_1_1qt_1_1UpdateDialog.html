Detailed Description

Modal dialog presenting available application and plugin updates.

Shows:

  • App section (hidden when no newer app version): version badge, release notes, "Open Release Page" button.
  • Plugin table: Name | Installed | Available | API compat | checkbox. Incompatible rows have their checkbox disabled.
  • "Update Selected" button downloads checked plugins via UpdateChecker, then signals MainWindow to apply them.
