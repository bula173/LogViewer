Detailed Description

template<typename FnType>
class plugin::FunctionRegistry< FnType >

Type-safe wrapper for dynamically loaded functions.

Provides compile-time type safety for C-style function pointers loaded from dynamic libraries via dlsym/GetProcAddress.

Template Parameters
FnTypeFunction pointer type (e.g., void(*)(int, const char*))
Usage Example
using CreatePanelFn = void*(*)(void*, void*, const char*);
FunctionRegistry<CreatePanelFn> creator(dllHandle, "Plugin_CreateLeftPanel");
if (creator) { // operator bool checks if symbol was found
auto* widget = creator.call(handle, parent, settings);
}
