Detailed Description

Information about a single plugin that has an available update.

The isCompatible flag is pre-computed by UpdateChecker based on the plugin manifest's apiVersion and minAppVersion fields. Incompatible updates are included in the result so the UI can display a warning, but UpdateDialog must not allow the user to install them.
