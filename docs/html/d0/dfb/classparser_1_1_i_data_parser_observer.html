Detailed Description

Observer interface for receiving notifications from data parsers.

This interface follows the Observer pattern to provide loose coupling between data parsing operations and user interface components. Observers receive notifications about parsing progress and newly discovered events.

Usage Example
class MyObserver : public IDataParserObserver {
public:
void ProgressUpdated() override {
// Update progress bar in UI
}
void NewEventFound(db::LogEvent&& event) override {
// Add event to display container
events.AddEvent(std::move(event));
}
};

virtual void parser::IDataParserObserver::NewEventBatchFound ( std::vector< std::pair< int, db::LogEvent::EventItems > > && eventBatch)
pure virtual

Called when a batch of new events is discovered during parsing.

For performance reasons, parsers may collect multiple events before notifying observers. This method delivers a batch of events at once.

Parameters
eventBatchVector of (id, items) pairs representing the events
Note
This method is preferred over NewEventFound for high-throughput scenarios
