Detailed Description

template<typename T, typename E>
class util::Result< T, E >

A type that represents either success (Ok) or failure (Err)

Result<T, E> is a modern C++ implementation of Rust's Result type. It forces explicit error handling at compile time and avoids the overhead of C++ exceptions in hot paths.

Template Parameters
TThe type of the success value
EThe type of the error value
Thread Safety:
Result is not thread-safe. Each thread should have its own instance.
Performance:
Zero overhead compared to manual error code checking when optimized. Uses std::variant internally for type-safe storage.
