const std::vector< int > &  rows,
const QString &  path 
)

Export the given view rows to a CSV file.

The first row of the output is a header derived from the model's horizontal header data. Each subsequent row corresponds to one entry in rows. Fields containing commas, double-quotes, or newlines are quoted per RFC 4180.

Parameters
modelThe item model (e.g. EventsTableModel) to read data from.
rowsView row indices to export (in order).
pathDestination file path.
Returns
true on success, false if the file could not be opened for writing.

Export the given view rows to an XML file.

Output is a well-formed XML document with a root <events> element. Each row becomes an <event> child; column values are stored as attributes. Attribute names are derived from the model's horizontal header data.
