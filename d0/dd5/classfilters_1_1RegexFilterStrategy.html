Detailed Description

Regex-based matching using std::regex.

Default strategy providing full regex support. Patterns are compiled once for efficiency.

Performance:
  • Compilation: O(m) where m = pattern length
  • Matching: O(n) where n = value length
  • Memory: Stores compiled regex state machine
