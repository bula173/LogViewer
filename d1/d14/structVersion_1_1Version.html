◆ asNumber()

long long Version::Version::asNumber ( ) const

Converts version to a single numeric representation.

Returns version in one big number with format: MMMMmmmmpppp (M-Major, m-minor, p-patch)

Example: Version 1.2.3 becomes 100020003 (leading zeroes are not displayed)

Returns
long long integer with encoded version information
Note
This method is useful for version comparison operations
