Detailed Description

Toolkit-agnostic controller interface for the MVC architecture.

Implementations coordinate between the data model (EventsContainer) and the view layer. The default implementation is mvc::MainController.

Note
The file that declares this interface is named IControler.hpp (one 'l') for historical reasons. Include that file, not this Doxygen-rendered name.
