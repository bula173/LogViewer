Detailed Description

Panel showing aggregate statistics per "actor" in the visible event set.

Two operating modes:

  • Defined-actor mode: when actor definitions (regexp) are supplied via SetDefinitions(), events are matched against those patterns and attributed to the first matching named actor.
  • Auto-detect mode (fallback): unique values of the selected field are treated as actors (original behaviour).

Clicking a row instantly filters the events view to that actor's events. "Clear Filter" resets to showing all visible events.
