Detailed Description

Asynchronous update checker using Qt6::Network.

Queries the GitHub Releases API for bula173/LogViewer and optionally downloads the plugins_manifest.json asset listed in that release to enumerate available plugin updates.

Usage:

auto* checker = new UpdateChecker(this);
connect(checker, &UpdateChecker::UpdateCheckComplete, this, &MyClass::OnUpdateResult);
checker->CheckAsync();
